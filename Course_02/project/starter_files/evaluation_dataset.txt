# NASA RAG System - Evaluation Dataset

This file contains sample questions used for testing the NASA RAG system, along with expected response characteristics and key information that should be included in accurate answers.

## Question 1: Apollo 11 Mission Overview

**Question:**
What was Apollo 11 and when did it take place?

**Expected Response Should Include:**
- Apollo 11 was the first crewed mission to land on the Moon
- Launch date: July 16, 1969
- Moon landing: July 20, 1969
- Crew members: Neil Armstrong (Commander), Buzz Aldrin (Lunar Module Pilot), Michael Collins (Command Module Pilot)
- Neil Armstrong and Buzz Aldrin walked on the Moon while Collins orbited
- Successful return to Earth on July 24, 1969

**Response Type:**
Factual historical summary with specific dates and crew names

## Question 2: Apollo 13 Crisis

**Question:**
What went wrong during the Apollo 13 mission?

**Expected Response Should Include:**
- Oxygen tank explosion occurred on April 13, 1970
- Famous quote: "Houston, we've had a problem"
- Crew: Jim Lovell, Jack Swigert, Fred Haise
- Service module oxygen tank No. 2 exploded
- Mission changed from Moon landing to safe crew return
- Crew used Lunar Module as "lifeboat"
- Successful splashdown on April 17, 1970
- Described as "successful failure"

**Response Type:**
Detailed incident description with problem-solving narrative

## Question 3: Challenger Disaster

**Question:**
What caused the Space Shuttle Challenger disaster?

**Expected Response Should Include:**
- Occurred on January 28, 1986
- 73 seconds after liftoff
- O-ring seal failure in right solid rocket booster
- Cold weather conditions contributed to failure
- Loss of all seven crew members
- Mission: STS-51-L
- Led to major safety reforms in NASA

**Response Type:**
Technical failure analysis with cause and consequences

## Question 7: Space Shuttle Program Context

**Question:**
What was the Space Shuttle program and what role did Challenger play?

**Expected Response Should Include:**
- Space Shuttle: Reusable spacecraft program (1981-2011)
- Challenger: Second operational orbiter (OV-099)
- First flight: April 4, 1983
- Completed 9 successful missions before disaster
- STS-51-L was 10th mission (January 1986)
- Disaster led to 32-month hiatus in shuttle flights
- Program continued until 2011 with improved safety

**Response Type:**
Historical context with program overview

## Question 8: Technical Specifications

**Question:**
What was the Lunar Module and what was its purpose?

**Expected Response Should Include:**
- Two-stage spacecraft designed for Moon landing
- Descent stage: Used for landing on Moon
- Ascent stage: Used for return to Command Module
- Could support two astronauts
- Separate from Command/Service Module
- Apollo 11's LM was named "Eagle"
- Apollo 13's LM "Aquarius" served as lifeboat
- Not designed for re-entry to Earth

**Response Type:**
Technical description with functional details

## Evaluation Metrics

When testing these questions, the system should demonstrate:

1. **Factual Accuracy**: Correct dates, names, and technical details
2. **Context Awareness**: Understanding of mission context and significance
3. **Source Attribution**: References to mission documents when available
4. **Completeness**: Coverage of key points listed in expected responses
5. **Relevance**: Staying on topic and avoiding irrelevant information
6. **Clarity**: Clear, well-organized responses
7. **Technical Precision**: Accurate use of technical terminology

## RAGAS Evaluation Expected Scores

- **Response Relevancy**: > 0.8 (responses should directly address the question)
- **Faithfulness**: > 0.7 (responses should be grounded in retrieved context)
- **Context Precision**: > 0.6 (retrieved documents should be relevant to question)

## Notes for Reviewers

- Actual responses will vary based on available documents in ChromaDB
- System should acknowledge when information is not available in context
- Responses should maintain professional, educational tone
- Technical details should be accurate when present in source documents
- System should cite sources when referencing specific mission transcripts or documents
